⌨️ Typing Speed Test: 

A clean, fast, and mobile-friendly typing speed test built with vanilla JavaScript.

🔗 Live Demo : [# https://mansityagi548.github.io/typing-test/]

📸 Preview : 

![Typing Speed Test Preview](preview/tying%20preview.png)

✨ Features:

⏱️ 60 second timed test — countdown timer with live stats

📊 Live WPM and Accuracy — updates as you type in real time

🎯 Three difficulty levels — Easy, Medium, and Hard passages

🏆 Personal best tracking — saved locally in your browser

🎉 Confetti celebration — fires when you beat your high score

📱 Mobile friendly — dropdown selectors and touch support

⌫ Backspace support — go back and fix mistakes

🖋️ Per-character feedback — green for correct, red for wrong

🛠️ Built With : 

HTML, CSS, JavaScript — no frameworks, no dependencies

Vanilla ES Modules

localStorage for personal best

🎮 How To Use : 

Choose a difficulty — Easy, Medium, or Hard

Click Start Typing Test or just start typing

Type the passage as fast and accurately as you can

See your WPM, Accuracy, and Characters when time runs out

Beat your Personal Best to trigger the confetti 🎉
